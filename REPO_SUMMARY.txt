USAGE WORKFLOW
--------------

For AI agents analyzing this repository:

1. Initial Context Load (2 min)
   - Read AI_NAVIGATION.md
   - Read analyses/ANALYSIS_INDEX.md
   - Review latest analysis in analyses/ directory

2. New Analysis (60-90 min)
   - Update config.json with target account
   - Run collection: python scripts/collect.py --limit 10
   - Run analysis: python scripts/analyze.py
   - Run reporting: python scripts/report.py
   - Review auto-outputs: data/processed/outputs/
   - Enrich top 25 accounts with web research
   - Generate strategic analysis documents
   - Save to analyses/YYYY-MM-DD-[target]/
   - Update ANALYSIS_INDEX.md

3. Strategic Planning (30-60 min)
   - Generate executive summary with top 25 lists
   - Map 5-7 strategic pathways
   - Create 30-day tactical engagement plan
   - Calculate ROI rankings
   - Identify critical success factors


TECHNICAL DETAILS
-----------------

Language: Python 3.14+
Dependencies: 8 core libraries (playwright, networkx, pandas, etc.)
Analysis Time: 30-60 minutes for 50-100 accounts
Test Coverage: ~75%

Collection Phases:
- Phase 0: Target profile metadata
- Phase 1: Post engagement (commenters, likers)
- Phase 2: Following list (often private)
- Phase 3: First-degree profiles

Analysis Algorithm:
- Multi-factor scoring: Proximity (40%) + Engagement (30%) + Bridge (20%) + Category (10%)
- Community detection: Greedy modularity
- Graph metrics: Betweenness, degree, closeness, eigenvector centrality

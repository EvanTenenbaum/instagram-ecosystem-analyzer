## Analysis Templates & Frameworks

### Standard Analysis Structure

Each analysis directory contains:

1. **Executive Summary** (`executive_summary_strategic_recommendations.md`)
   - Key findings (single most important discovery)
   - Top 25 lists (galleries, collectors, designers, press, platforms)
   - Strategic pathways (ranked by probability and timeline)
   - Expected ROI rankings
   - Critical success factors
   - 12-month objectives

2. **Strategic Pathways Analysis** (`strategic_pathways_analysis.md`)
   - Detailed pathway mapping
   - Hidden opportunities discovered
   - Graph analysis (centrality metrics, clustering)
   - Risk assessment
   - Timeline-based recommendations
   - Ecosystem clustering analysis

3. **Tactical Engagement Plan** (`engagement_plan_30_days.md` / `engagement_plan_90_days.md`)
   - Day-by-day or week-by-week actions
   - Specific comment examples
   - Physical event attendance strategy
   - Success metrics
   - Red flags to avoid

4. **Account Intelligence** (`top_100_accounts_enriched.csv`)
   - Account classifications
   - Career leverage scores
   - Introduction power ratings
   - Collector/gallery/architecture access scores
   - Recommended actions per account

### Analysis Types

**Type 1: Gallery Representation Pathway**
- Target: Specific gallery or gallery network
- Objective: Identify fastest routes to representation
- Output: Bridge nodes, warm paths, engagement strategies
- Example: Sarah Myerscough analysis

**Type 2: Collector Access Mapping**
- Target: Fair, auction house, or collector community
- Objective: Map collector network and access points
- Output: Collector-adjacent accounts, advisor network, designer connections

**Type 3: Curator Visibility Strategy**
- Target: Museum or institutional ecosystem
- Objective: Identify curatorial network
- Output: Curator accounts, board members, acquisition influencers

**Type 4: Commission Pipeline Building**
- Target: Interior designer or architecture firm networks
- Objective: Map designer relationships for commission opportunities
- Output: Designer contacts, their collectors, project types

**Type 5: Press Coverage Pathways**
- Target: Design publications (AD, Elle Decor, etc.)
- Objective: Identify journalist/editor networks
- Output: Press contacts, gallery relationships, feature opportunities

**Type 6: Cross-Network Analysis**
- Target: Multiple related accounts (e.g., 5 galleries in same sector)
- Objective: Find universal bridge nodes and shared communities
- Output: Super-connectors who span multiple target ecosystems
